Treatment Recommender Widget Integration

How This Works

This example shows how to embed the Treatment Recommender as an iframe. The widget is hosted on GitHub Pages, and you only need to add a simple iframe to your website.

Benefits:

  • No JavaScript or CSS files on your server
  • No code blocks in WordPress
  • Widget updates happen automatically
  • Easy to implement for clients with any website platform

Live Demo

Embed Code

Copy and paste this code into your website where you want the widget to appear:

<!-- Treatment Recommender Widget -->
<iframe 
  id="treatment-recommender-widget" 
  src="https://vitalityventures.github.io/treatmentrecommender/iframe-widget.html?title=YOUR%20CLINIC%20NAME&subtitle=Custom%20subtitle&serviceId=YOUR_SERVICE_ID&templateId=YOUR_TEMPLATE_ID&userId=YOUR_USER_ID"
  style="width: 100%; min-height: 750px; border: none;" 
  title="Treatment Recommender Widget"
  allow="clipboard-write"
></iframe>

<script>
  // Automatically resize iframe based on content height
  window.addEventListener('message', function(event) {
    if (event.data && event.data.type === 'resize') {
      const iframe = document.getElementById('treatment-recommender-widget');
      if (iframe && event.data.height) {
        iframe.style.height = (event.data.height + 50) + 'px';
      }
    }
  });
</script>

Configuration Options

Customize the widget by adding these URL parameters:

  • serviceId - EmailJS service ID
  • templateId - EmailJS template ID
  • userId - EmailJS user ID
  • title - Clinic/brand name
  • subtitle - Custom subtitle
  • copyright - Copyright name
  • showHeader - Set to 'true' to show header (default: false)
  • showFooter - Set to 'true' to show footer (default: false)
