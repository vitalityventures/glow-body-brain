How to Integrate the Treatment Recommender Widget

Step 1: Set Up EmailJS

Create an account at EmailJS and set up a service and email template. Your template should include parameters like: {firstName}, {lastName}, {email}, {phone}, {newsletter}, {treatmentPlan}, {clientSiteInfo}, {companyName}, and {clientSiteName}.

Step 2: Include the Treatment Recommender Script

Add the compiled JavaScript to your HTML:

<script src="path/to/treatment-recommender.js"></script>

Step 3: Configure EmailJS

Configure the EmailJS service with your company credentials and client information:


<script>
  // Set up EmailJS configuration
  window.TreatmentRecommender.configureEmailService({
    serviceId: 'YOUR_EMAILJS_SERVICE_ID',
    templateId: 'YOUR_EMAILJS_TEMPLATE_ID',
    userId: 'YOUR_EMAILJS_USER_ID',
    recipientEmail: 'your-company@example.com',
    companyName: 'Your Company Name',
    clientSiteName: 'Client Beauty Clinic'
  });
</script>

Step 4: Customize the Widget for Each Client

Customize the widget appearance and content for each client:


<script>
  // Customize for this specific client
  window.TreatmentRecommender.configureSiteSettings({
    title: "CLIENT NAME BEAUTY",
    subtitle: "Find your perfect treatment",
    aboutUsText: "Client's custom about text here.",
    contactInfo: {
      email: "contact@clientdomain.com",
      phone: "(555) 555-5555",
      address: "123 Client Street, City",
    },
    navigation: {
      showAbout: true,
      showTreatments: true,
      showIntegrationGuide: false, // Hide for client sites
    },
    copyrightName: "Client Name Beauty",
  });
</script>

Step 5: Add the Treatment Recommender to Your Page

Insert the recommender widget in your desired location:


<div id="treatment-recommender-container"></div>

<script>
  // Render the Treatment Recommender component
  ReactDOM.render(
    React.createElement(window.TreatmentRecommender.default),
    document.getElementById('treatment-recommender-container')
  );
</script>

Setting Up Your Company's Email Workflow

For Agency/Company Use: This configuration allows you to centralize all client leads through your company email first.

  1. Use Your Company's EmailJS Account: Create one account that you'll manage for all clients.
  2. Create an Email Template: Design a template that includes all treatment plan details and client information.
  3. Configure Auto-Forwarding: Set up your email system to automatically forward submissions to the appropriate client based on the clientSiteName field.
  4. Template Variables: Make sure your EmailJS template includes all fields, including {clientSiteInfo}, {companyName}, and {clientSiteName}.

This approach gives you complete control over the lead flow while maintaining a custom experience for each client site.

Note: With this setup, your company will receive all form submissions first, which you can then forward to the respective clients. This gives you oversight on all leads and allows you to provide additional value to your clients while maintaining the relationship.
