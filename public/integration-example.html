How to Integrate the Treatment Recommender Widget

Step 1: Set Up EmailJS

Create an account at EmailJS and set up a service and email template. Your template should include parameters like: {firstName}, {lastName}, {email}, {phone}, {newsletter}, and {treatmentPlan}.

Step 2: Include the Treatment Recommender Script

Add the compiled JavaScript to your HTML:

<script src="path/to/treatment-recommender.js"></script>

Step 3: Configure EmailJS

Configure the EmailJS service with your credentials:


<script>
  // Set up EmailJS configuration
  window.TreatmentRecommender.configureEmailService({
    serviceId: 'YOUR_EMAILJS_SERVICE_ID',
    templateId: 'YOUR_EMAILJS_TEMPLATE_ID',
    userId: 'YOUR_EMAILJS_USER_ID',
    recipientEmail: 'admin@yourdomain.com'
  });
</script>

Step 4: Customize the Widget (Optional)

Customize the widget appearance and content for each client:


<script>
  // Customize for this specific client
  window.TreatmentRecommender.configureSiteSettings({
    title: "CLIENT NAME BEAUTY",
    subtitle: "Find your perfect treatment",
    aboutUsText: "Client's custom about text here.",
    contactInfo: {
      email: "contact@clientdomain.com",
      phone: "(555) 555-5555",
      address: "123 Client Street, City",
    },
    navigation: {
      showAbout: true,
      showTreatments: true,
      showIntegrationGuide: false, // Hide for client sites
    },
    copyrightName: "Client Name Beauty",
  });
</script>

Step 5: Add the Treatment Recommender to Your Page

Insert the recommender widget in your desired location:


<div id="treatment-recommender-container"></div>

<script>
  // Render the Treatment Recommender component
  ReactDOM.render(
    React.createElement(window.TreatmentRecommender.default),
    document.getElementById('treatment-recommender-container')
  );
</script>

Full Example


<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Client Beauty Clinic</title>
  
  <!-- React Dependencies -->
  <script src="https://unpkg.com/react@18/umd/react.production.min.js"></script>
  <script src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script>
  
  <!-- Treatment Recommender Widget -->
  <script src="path/to/treatment-recommender.js"></script>
  <link rel="stylesheet" href="path/to/treatment-recommender.css">
</head>
<body>
  <h1>Client Beauty Clinic</h1>
  
  <div id="treatment-recommender-container"></div>
  
  <script>
    // Configure EmailJS
    window.TreatmentRecommender.configureEmailService({
      serviceId: 'YOUR_EMAILJS_SERVICE_ID',
      templateId: 'YOUR_EMAILJS_TEMPLATE_ID',
      userId: 'YOUR_EMAILJS_USER_ID',
      recipientEmail: 'contact@clientdomain.com'
    });
    
    // Customize for this specific client
    window.TreatmentRecommender.configureSiteSettings({
      title: "CLIENT BEAUTY",
      subtitle: "Discover your perfect treatment at Client Beauty Clinic",
      aboutUsText: "Client Beauty Clinic offers premium aesthetic services.",
      contactInfo: {
        email: "contact@clientdomain.com",
        phone: "(555) 987-6543",
        address: "123 Beauty Street, City, State",
      },
      navigation: {
        showAbout: true,
        showTreatments: true,
        showIntegrationGuide: false,
      },
      copyrightName: "Client Beauty Clinic",
    });
    
    // Render the widget
    ReactDOM.render(
      React.createElement(window.TreatmentRecommender.default),
      document.getElementById('treatment-recommender-container')
    );
  </script>
</body>
</html>

Note: For agencies and service providers, you can create a custom configuration for each client, including setting up EmailJS on their behalf. Just make sure each client has their own unique EmailJS account for email quota management.
