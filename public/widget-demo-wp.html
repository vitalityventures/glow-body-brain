Treatment Recommender Widget for WordPress

This page demonstrates how to embed the treatment recommender widget in WordPress.

WordPress Integration Checklist

  1. Upload the UMD JavaScript file
  2. Upload the CSS file
  3. Add a Code Block to a WordPress page
  4. Add the code below, adjusting file paths to your uploads

Important for WordPress

For WordPress integration, you MUST use the UMD version of the file. The regular ESM version will not work because WordPress does not support ES modules.

Use the file with .umd.js extension in your uploads.

WordPress Code Block / HTML Block

Add the following code to your WordPress page using either a Code Block or HTML Block:

<!-- React Dependencies --> <script src="https://unpkg.com/react@18/umd/react.production.min.js" crossorigin></script> <script src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js" crossorigin></script> <!-- Widget Files - Update these URLs with your actual WordPress media URLs --> <script src="https://example.com/wp-content/uploads/2023/04/treatment-recommender.umd.js"></script> <link rel="stylesheet" href="https://example.com/wp-content/uploads/2023/04/style.css"> <!-- Widget Container --> <div id="treatment-recommender-container" style="width: 100%; min-height: 800px;"></div> <script> // Configure EmailJS window.TreatmentRecommender.configureEmailService({ serviceId: "your_emailjs_service_id", templateId: "your_emailjs_template_id", userId: "your_emailjs_user_id" }); // Configure Widget Settings window.TreatmentRecommender.configureSiteSettings({ title: "YOUR BRAND NAME", subtitle: "Your custom subtitle here", copyrightName: "Your Brand", layout: { showHeader: true, showFooter: true } }); // Render the widget document.addEventListener('DOMContentLoaded', function() { var container = document.getElementById('treatment-recommender-container'); if (window.ReactDOM.createRoot) { ReactDOM.createRoot(container).render( React.createElement(TreatmentRecommender.default) ); } else { ReactDOM.render( React.createElement(TreatmentRecommender.default), container ); } }); </script>

WordPress with Divi Builder

If you're using the Divi Builder, follow these additional steps:

  1. Create a new page or edit an existing one
  2. Add a "Code Module" element
  3. Paste the code above into the module
  4. Important: Make sure "Disable on-page builder" is checked in the module settings
  5. Save and publish the page

Troubleshooting

If the widget doesn't appear:

  1. Check your browser console for errors (F12 or right-click → Inspect → Console)
  2. Verify that your JavaScript and CSS file URLs are correct
  3. Make sure you're using the UMD version of the JavaScript file, not the ESM version
  4. If using Divi, make sure the "Disable on-page builder" option is checked
  5. Try a different browser to rule out browser-specific issues
