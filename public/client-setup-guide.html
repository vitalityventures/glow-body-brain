Treatment Recommender - Client Setup Guide

Quick Start: Add to WordPress

To add the Treatment Recommender to a WordPress site:

  1. Go to the page where you want to add the widget
  2. Add a "Custom HTML" block
  3. Paste the iframe code (see examples below)
  4. Update the parameters for your specific client
  5. Save and publish the page

Basic WordPress Example

<iframe 
  src="https://yourusername.github.io/treatment-recommender/iframe-widget.html?serviceId=YOUR_EMAILJS_SERVICE_ID&templateId=YOUR_EMAILJS_TEMPLATE_ID&userId=YOUR_EMAILJS_USER_ID&title=CLINIC_NAME" 
  width="100%" 
  height="800px" 
  frameborder="0"
  allow="web-share"
></iframe>

All Available URL Parameters

Customize the widget for each client by changing these URL parameters:

Parameter Description Example
serviceId EmailJS service ID service_abc123
templateId EmailJS template ID template_xyz789
userId EmailJS user ID user_def456
title Clinic or brand name Serenity Med Spa
subtitle Subtitle text Your Beauty Journey Starts Here
copyright Copyright name in footer Serenity Med Spa
showHeader Show or hide header (true/false) false
showFooter Show or hide footer (true/false) false
Note: URL parameters should be properly encoded, especially if they contain spaces or special characters.

Setting Up EmailJS

For each client, you'll need to set up EmailJS:

  1. Create an account at EmailJS.com
  2. Add an email service (like Gmail, Outlook, etc.)
  3. Create an email template with these variables:
    • firstName
    • lastName
    • email
    • phone
    • newsletter
    • treatmentPlan
    • timestamp
    • source
  4. Note your service ID, template ID, and user ID

Troubleshooting

  • Widget not loading: Check if the iframe URL is correct and the repository is properly deployed.
  • Emails not sending: Verify the EmailJS credentials and check console for errors.
  • Layout issues: Try adjusting the iframe height or width.
